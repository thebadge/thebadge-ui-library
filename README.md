## Develop the library

### Run storybook

```
yarn storybook
```

### Run tests

```
yarn test
```

### Check lint

```
yarn lint
```

### Fix lint errors

```
yarn lint:fix
```

## Integrate the library in your project

```
yarn add thebadge-ui-library
```

Import CSS files with styles in your \_app.tx file

```
import 'node_modules/thebadge-ui-library/dist/index.css'
```

## Build it

```
yarn build
```
